Summary

This html is meant to reflect the change in weather patterns influenced by proximity to the equator. Data depicted is sourced from the Open Weather API through the CityPy Library in Python. 500 cities were pulled at random from around the world to create the visualizations.

Analysis of the intial data set was completed using Pandas and Matplotlib Libraries in Python. The elements analyzed included Temperature, Humidity, Cloudiness, and wind Speed.
